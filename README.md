# 💸 Expense Tracker

A clean, interactive web app to track your income and expenses in real time. Built with vanilla JavaScript, HTML, and CSS — no frameworks, no dependencies.

## 📌 Overview

A personal finance tracker that helps users log transactions (income or expense), view their balance at a glance, and delete entries. All data is persisted in `localStorage` so it survives page refreshes.

---

## ✨ Features

- ➕ Add income and expense transactions
- 💰 Live balance, total income, and total expense display
- 🗑️ Delete any transaction
- 💾 Data persisted via `localStorage`
- 📱 Responsive design

---

## 🚀 How to Run

No setup needed — just open in your browser!

```bash
git clone https://github.com/glakshya20/Expense-Tracker-Project.git
cd Expense-Tracker-Project
# Open index.html in your browser
```

Or simply double-click `index.html`.

## 📸 How It Works

1. Enter a transaction name and amount
2. Use a **positive value** for income, **negative** for expense
3. Click **Add Transaction**
4. Your balance updates instantly
5. Click ✖ next to any item to delete it

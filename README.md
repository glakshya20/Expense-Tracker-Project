# 💸 Expense Tracker App

🔗 **Live Demo:** https://trackyourexpensesfast.netlify.app/  

---

## 🚀 Overview

A modern and responsive **Expense Tracker Web App** built using React that helps users manage their income and expenses efficiently.  
Track your spending, monitor balance, and stay financially aware — all in one place.

---

## ✨ Features

- ➕ Add income & expense transactions  
- ➖ Delete transactions  
- 💰 Real-time balance calculation  
- 📊 Income vs Expense summary  
- 💾 Data persistence using Local Storage  
- ⚡ Fast and responsive UI  

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Styling:** CSS  
- **State Management:** React Hooks (useState, useContext)  
- **Storage:** Local Storage  
- **Deployment:** Netlify

## ⚙️ Installation & Setup

```bash
git clone https://github.com/glakshya20/Expense-Tracker-Project.git
cd Expense-Tracker-Project
```

```bash
npm install
```

```bash
npm start
```

## 📦 Build for Production

```bash
npm run build
```
